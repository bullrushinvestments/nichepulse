# NichePulse

NichePulse is an AI-driven subscription box service tailored for small businesses in the creator economy, providing curated content and tools to enhance their brand presence and community engagement.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy to Vercel:
```bash
vercel deploy --prod
```
